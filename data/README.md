# Datos

Este proyecto utiliza distintas fuentes de datos para el análisis del riesgo asociado a incendios forestales.

## Fuentes utilizadas:

- Incendios Forestales.
- Postas Rurales.
- Datos Meteorológicos (Open-Meteo).

## Importante:

Los archivos originales no se incluyen en este repositorio debido a su tamaño y a las condiciones de disponibilidad de las fuentes de datos.

Las transformaciones y procesos ETL se encuentran documentados en las carpetas `api` y `sql`.
